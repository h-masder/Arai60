## step0
問題を読み、以下のように実装すればよいと考えた。
・linked listがcycleを持つことの確認は、たどったノードを覚えておき、同じノードにたどりつくか判定すればよい。
・linked listの構造が

```
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None
```

と定義されていたため、たどっていった先のノードが`None`になるまで、上記の判定を行えばよいと考えた。

Pythonの書き方をほとんど知らないため、ここで答えを確認した。

そちらにあるコードを参考にして実装した。
ただし、以下の2点はコードが読みにくくなっていると感じたため、修正した。
1) たどったノードを記録する変数が`dict`だが`value`が使われていない。使っていない値があると読みにくいと感じたため、`set`に変更した。
2) `while`文で条件の判定対象となっているノード（`node.next`）と、`while`の内部で判定対象となっているノード（`node`）が一致していない。この不一致で読むのに時間がかかったため、双方が一致するように修正した。
